To create this program, go to the localhost folder and set up an express backend app 

First install express generator 
	$npm install -g express-generator 

Create the express app 
	$express PROJECT_NAME 
	$cd PROJECT_NAME 
	$npm install 

Install concurrently 
	$npm install concurrently 

Create the client within 
	$create-react-app client 

Install body parser 
	$npm install body-parser

Install mysql 
	$npm install mysql

Modify /routes/users.js to be the server (modify filename in /app.js)

Modify package.json for "start" command 

Modify /client/src/App.js, /client/src/Form.js, and /client/public/index.html, remove irrelevant files 

MySQL need phpmyadmin or MySQL workbench to setup
